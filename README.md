# bash

## 📋 Descrição

Repositório contendo scripts bash e configurações para inicializar e gerenciar o projeto **LogiTrack**.

---

## 🚀 Quick Start - Inicializar Repositório Git

Execute os comandos abaixo em sequência para configurar o repositório:

```bash
git init
git add .
git commit -m "Initial commit: Estrutura LogiTrack"
git branch -M main
git remote add origin https://github.com/vazdemedeirossidney-ops/logitrack-app.git
git push -u origin main
```

---

## 📖 Detalhamento dos Comandos

| Comando | Descrição |
|---------|-----------|
| `git init` | Inicializa um novo repositório Git local |
| `git add .` | Adiciona todos os arquivos ao staging area |
| `git commit -m "..."` | Cria um commit com mensagem descritiva |
| `git branch -M main` | Renomeia a branch padrão para "main" |
| `git remote add origin ...` | Configura o repositório remoto |
| `git push -u origin main` | Envia commits para o repositório remoto |

## ⚙️ Pré-requisitos

- Git 2.28+ instalado
- Acesso SSH ou HTTPS ao GitHub

---

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/vazdemedeirossidney-ops/bash.git
cd bash
```

2. Execute o script de setup:
```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```
